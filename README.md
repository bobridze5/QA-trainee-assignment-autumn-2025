## Задание №2.1
Данный проект содержит автоматизированные тесты для проверки API микросервиса:
https://qa-internship.avito.com

В тестах покрыты следующие ручки:

1. Создание объявления
2. Получение объявления по его идентификатору
3. Получение всех объявлений по идентификатору продавца
4. Получение статистики по itemId

Тесты написаны на основании тест-кейсов, оформленных в файле TESTCASES.md.  
Ошибки, обнаруженные в процессе тестирования, отражены в BUGS.md.

---

### Используемые технологии

Проект реализован на **Java 21.**  
Используются следующие библиотеки и инструменты:

- TestNG — фреймворк для написания автотестов
- REST Assured — библиотека для тестирования REST API
- Gson — сериализация и десериализация JSON
- Lombok — сокращение шаблонного кода моделей
- Maven — управление зависимостями и запуск тестов
- Docker — контейнеризация тестового окружения (опционально)

---

### Запуск тестов локально

Требования:
- Java 21+
- Maven 3.8+

Команда для запуска всех тестов:

```bash
cd Task2.1/

mvn clean test
```

## С помощью Docker

```bash
docker build -t task-tests .

docker run task-tests
```
